# cay 22 岁生日快乐 🎂✨

一个可以直接打开的小网站：
- 居中旋转的像素风发光蛋糕
- 点击后触发烟花、显示两张照片
- 背景音乐为“生日快乐”旋律（WebAudio 合成）

## 使用方法

1. 直接双击打开 `index.html` 即可在浏览器查看。
2. 替换照片：
   - 将你们的真实照片放入 `assets/photos/`，并更新 `index.html` 中两张图片的路径，或覆盖 `me.svg`、`her.svg`。
3. 若需本地服务器（有时更兼容媒体播放）：

```bash
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 交互说明
- 点击蛋糕会触发：
  - 烟花在背景绽放
  - 两张照片出现（支持小屏自适应）
  - 背景播放“生日快乐”旋律（需用户首次交互，符合浏览器自动播放策略）

## 自定义
- 如需更换配色、发光强度或旋转速度，可编辑 `css/style.css`。
- 如需调整烟花数量、速度，可在 `js/main.js` 中修改 `burst`、`setInterval` 等参数。
- 如需改为使用本地 MP3，请在 `index.html` 中增加 `<audio>` 标签并在点击时播放，或将 WebAudio 逻辑替换为文件播放。
